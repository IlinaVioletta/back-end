# Проект: Сайт гостьової книги

## Опис
Це простий вебдодаток на PHP, який реалізує гостьову книгу з можливістю реєстрації, входу та додавання коментарів.

Проєкт побудований з використанням ООП та архітектури MVC (Model-View-Controller).

---

## Функціонал
- Реєстрація користувача
- Авторизація (вхід)
- Вихід з акаунту
- Додавання коментарів
- Перегляд коментарів

---

## Технології
- PHP
- MySQL
- PDO
- Composer

---

## Структура проєкту
- `Controllers/` - контролери (логіка)
- `Models/` - робота з базою даних
- `Views/` - відображення (HTML)
- `index.php` - єдина точка входу
- `config.php` - налаштування бази даних

---

## Запуск проєкту
1. Встановити залежності: `composer install`
2. Налаштувати підключення до БД у файлі `config.php`
3. Запустити сервер: `php -S localhost:3000`
4. Відкрити в браузері: http://localhost:3000
